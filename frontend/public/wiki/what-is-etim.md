# What Is ETIM?

`ETIM` is een gestandaardiseerd classificatiemodel voor technische producten.

In deze applicatie gebruiken we ETIM als:

- semantische basis voor productklassen
- bron voor features
- bron voor units en mogelijke waarden

## Belangrijk

ETIM is **niet automatisch de uiteindelijke invoervorm**.

Daarom bouwen we er een extra engineeringlaag bovenop:

- governed parameters
- local parameters
- mappings
- instances
